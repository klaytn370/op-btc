# Op-BTC (TrustFund)

TrustFund BTC is a shared fund application concept on Bitcoin Layer 1 using OPNet.

## Problem
Many groups want to collect money together, but they don't trust one person to hold the funds.

## Solution
A shared fund using Bitcoin and OPNet smart contracts with transparent balance and rules-based withdrawals.

## Status
Prototype for OP_NET Vibecoding challenge.
